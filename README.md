# Owl

Owl is a command line interface to interact with LDAP.

## Example

```bash
owl login ${server} -u ${user}
owl use ${basedn}
owl get users
```

## Installation

Download the latest version for your OS from the [release page](https://github.com/adrienaury/owl/releases).
